# [Shortener Cat API]

An URL shortener with a simple RESTful API.

## Introduction

This API (cat) goal is to shorten your long urls while giving you analytics and caching features ðŸ˜¸.

## Some Features

-   URL analytics (ex: max visits by country).
-   URL caching using redis.
-   Push notifications using FCM.
-   Authentication, either by creating a new account or using google with OAuth 2.0.
-   Forgot and reset password.
-   Email sending.
-   Swagger documentation and API versioning.
-   Docker compose for easy and quick setup.
-   Unit tests using XUnit and Moq

## Getting Started

To get started with this project, follow these steps:

### Prerequisites

-   Docker

### Installation

1. **Clone the Repository (or just download the docker-compose.yml file)**

    ```bash
    git clone https://github.com/youssef-rafie212/Shortener-Cat
    ```

2. **Navigate to the project directory**

3. **Run the docker-compose file**

    ```bash
    docker-compose up -d
    ```

### Running The Application

**After running the docker-compose file a composed container will start running on port 6969**
